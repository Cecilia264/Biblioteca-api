# Library Management API

Proyecto personal desarrollado para practicar Backend utilizando:

- Python
- FastAPI
- MySQL
- SQL
- Postman
- Git/GitHub

## Funcionalidades

- Gestión de usuarios
- Gestión de libros
- Gestión de categorías
- Gestión de autores
- Gestión de préstamos

Proyecto desarrollado con fines de aprendizaje.
